Hello Electron!

This is a simple Hello World app for electron js.

Press Ctrl+Shift+I to see the console.

The source code for this app is available at https://github.com/yozi66/hello-electron

How to start this app?

You have several choices:

  • Open index.html in a browser. This will not run the "electron" code, just the embedded script of this page.
  • npm run start
    from the command line.
  • Open package.json in VS Code and click "Debug" above the "Scripts" line.
  • VS Code uses the 
    .vscode/launch.json
    file for the "Run" menu (F5, Ctrl+F5) and the "Run and Debug" view (Ctrl+Shift+D).
